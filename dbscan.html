What is DBSCAN ?

In density based clustering, data is grouped by areas of high concentrations of data points surrounded by areas of low concentrations of data points. Basically the algorithm finds the places that are dense with data points and calls those clusters. It can discover clusters of different shapes and sizes from a large amount of data, which is containing noise and outliers. DBSCAN clustering's most appealing feature is its robustness against outliers.
This Algorithm requires only two parameter namely min-Points(minpts) and epsilon(eps).
* minpts - The minimum number of points (a threshold) clustered together for a region is min-point
* epsilon - the distance measure that will be used to locate the points in the neighborhood of any point.
DBSCAN creates a circle of epsilon radius around every data point and classifies them into Core point, Border point, and Noise.
In this algorithm, we have 3 types of data points.
   1. Core point       - Points that satisfy minpts condition.
   2. Boundary points  - Points that does not have sufficient minpts but it is coming under a core point.
   3. Outlier / Noise   - Point which is neither core point and boundary point.

Algorithm:

  1. Choose minpts and eps value
  2. Arbitarly pick up all points in the dataset one by one.
  3. Select the region with the help of eps value
  4. If the point satisfy minpts then create new cluster
  5. Else marked 'p' is visited(p-point)
  6. If cluster contain more than one point or 'p' is a part of any cluster then add to boundary point
  7. Else add to noise

Advantages:

  • Does not require a-priori specification of number of clusters.
  • Able to identify noise data while clustering.
  • DBSCAN algorithm is able to find arbitrarily size and arbitrarily shaped clusters.
  • Produce various number of clusters
  • If we don't know number of clusters and there is no way to visualize our dataset then can use DBSCAN algorithm
