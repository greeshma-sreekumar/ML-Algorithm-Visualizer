What is K-Means ?

K-Means Clustering is an Unsupervised Learning Algorithm, which groups the unlabeled dataset into different clusters. It's a partition based technique or centroid based technique. Segment database into groups based on similarity of datapoints. K-Means performs the division of objects into clusters that share similarities and are dissimilar to the objects belonging to another cluster. The main goal is to divide N observations into K clusters repeatedly until no more groups can be formed The algorithm takes the unlabeled dataset as input, divides the dataset into k-number of clusters, and repeats the process until it does not find the best clusters. The value of k should be predetermined in this algorithm.
The k-means clustering algorithm mainly performs two tasks:
   1.Determines the best value for K center points or centroids by an iterative process.
   2.Assigns each data point to its closest k-center. Those data points which are near to the particular k-center, create a cluster.

Hence each cluster has data points with some commonalities, and it is away from other clusters.

Algorithm:

  1. Intitialize the clusters(number of clusters,k) and centroid value
  2. Select the next object and find Euclidean distance from these clusters,
    if object (xi,yj) and one cluster (xa,yb) then Euclidean distance, d=√[(xi-xa)²-(yj-yb)²]
  3. Assign object to the cluster which having lesser Euclidean distance
  4. Update centroid value
  5. Repeat the process untill all point are included

Advantages:

  • It is very easy to understand and implement.
  • If we have large number of variables then, K-means would be faster than density based algorithm
  • On re-computation of centroids, an instance can change the cluster.
  • Tighter clusters are formed with K-means as compared to density based algorithm
  • we can use large datatsets for clustering
